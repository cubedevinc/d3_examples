# Cube AI Agent Integration Examples

Examples demonstrating how to integrate Cube AI Agents into your applications using the Chat API and workflow orchestration.

## Examples

- **[shared/cube-agent-client/](shared/cube-agent-client/)** - Reusable Node.js client library for Cube AI Agents
- **[chat-api/](chat-api/)** - Simple chat API example with streaming responses
- **[langgraph-analytics/](langgraph-analytics/)** - Advanced workflow orchestration with LangGraph.js
- **[react-app/](react-app/)** - Next.js React application with chat interface

See each folder's README for detailed setup and usage instructions.

## Environment Variables

```bash
CUBE_TENANT_NAME=your-tenant-name
CUBE_AGENT_ID=1
CUBE_API_KEY=your-api-key
```
